## How to start the project ?

### Development mode

`docker-compose -f docker-compose.dev.yml up --build -d`

### Production mode

`docker-compose up --build -d`


### client mongo

`docker-compose exec my-mongoclient /bin/sh`


### Use the app

- The app is running at : `localhost:8000`
